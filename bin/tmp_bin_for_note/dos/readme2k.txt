1. Windows 2000を導入する前の準備
---------------------------------

   (1) 必要なデータはバックアップしておく。

       Windows 2000のインストールを開始する前に、万一の場合に備えて必要なデータ
       はバックアップをとっておいてください。

   (2) BIOSを最新のバージョンにアップデートしておく。

        == BIOSとは ==
       パソコンのハードウェアを制御する基本的なソフトウェアです。
       パソコンの機種によっては、Windows 2000導入時にBIOSのアップデートが必要な
       場合があります。ご使用のパソコンについて、製造元のWebサイトなどで最新の
       BIOSバージョンをご確認頂き、必要であればBIOSのアップデートを実施して
       ください。なお、BIOSのアップデートを行う前には、ヘルプ等をよく読んでから
       実施してください。

   (3) トラブルに備えて、Windows 2000用のセットアップ起動ディスクを作成する。
       (フォーマット済み空フロッピーディスクが4枚必要です)

       このフロッピーディスクは、Windows 2000をインストールする際　万一トラブル
       が発生した場合などに利用する重要なディスクです。
       == 作成方法 ==
       ① Windows 2000のCD-ROMをドライブに挿入する。
       ② ｢スタート｣-｢ファイル名を指定して実行｣をクリックする。
       ③ ｢名前｣欄に半角で｢d:\bootdisk\makeboot.exe｣と入力し、｢OK｣をクリックする。
           (dは、実際のCD-ROMドライブのドライブ名) 
       ④ ｢コピー先のフロッピーディスクを指定してください｣に対して、フロッピー
          ディスクドライブ名を半角で入力する。
       ⑤ FDドライブに、空フロッピーディスクを挿入し｢Enter｣キーを押す。
       ⑥ 画面に表示される指示に従って操作する。

2. Windows 2000 インストール方法について
-----------------------------------------

       Windows 2000をインストールするには、次の３種類の方法があります。

        ① Windows 98SE/Me上から、新規にWindows 2000をインストールする
           既存のOS環境はそのままで、新規にWindows 2000をインストールし1台の
           パソコンで｢デュアルブート」させることができます。

        ② Windows 98SE/MeからWindows 2000にアップグレードする(上書きインストール)
           Windows 98SE/Meの既存の環境をそのまま引き継ぐインストール方式です。
           具体的には、現在使用中のOS環境で設定した各種項目(マシン名、ワーク
           グループ/ドメインの設定、インストールしたアプリケーションやそこで
           設定したパラメータなど)が引き継がれます。しかし、インストールに失敗
           すると、元の環境をすべて失ってしまう危険性があります。

        ③ ハードディスクをフォーマット後、新規にWindows 2000をインストール
           OSがインストールされていないパソコン、すなわちハードディスクを
           フォーマットして新規にWindows 2000をインストールする方法です。

　　　== お願い ==
      内蔵CD-ROM/DVD-ROMドライブがあるパソコンで、Windows 2000をインストール
      される場合には、なるべく本機でなく内蔵ドライブを使ってWindows 2000の
      インストールを行ってください。

3. Windows 98SE/Me上からWindows 2000をインストールする
-------------------------------------------------------

  Windows 98SE/Me上から、Windows 2000を新規にインストールする(デュアルブート
  環境の構築)、またはWindows 2000にアップグレードする場合には、次の方法で実施
  してください。

  == お知らせ ==

  ・Windows 2000のCD-ROMをセットして、セットアッププログラムが起動した場合は、
    セットアッププログラムを終了してください。

　・以下の方法にて、インストールを進めていくと途中で｢プラグアンドプレイの更新
    されたファイルの提供｣画面が現れ、この中で本機用のデバイスドライバーも表示
    されていても、そのまま｢次へ｣をクリックして先へ進めてください。

  ・コンピュータ再起動後ブルー画面で以下のメッセージが表示された場合には、
    Enterキーで先へ進めてください。

  Windows 2000 Setup
  ==================
  Setup could not determine the type of one or more mass storage devices
  installed in your system, or you have chosen to manually specify an adapter.
  Currently, Setup will load support for the following mass storage devices(s):

     <none>


    == インストール方法 == 

    (1) [スタート]をクリックし、[ファイル名を指定して実行]をクリックする。

    (2) [名前]に、以下のように入力し、[OK]ボタンをクリックする。
         (CDドライブ名):\i386\winnt32.exe /makelocalsource

        /makelocalsourceは、インストールファイルをすべてハードディスクへコピー
        させるためのパラメータです。

    (3) ｢Windows 2000 セットアップウィザードの開始｣画面が表示されますので、
        画面に表示される指示に従ってインストールする。

4. ハードディスクをフォーマット後、新規にWindows 2000をインストールする
-----------------------------------------------------------------------

       ハードディスクをフォーマットする前に、ご使用のパソコンで本機が起動する
       ように、CD-ROM起動ディスクを作成しておく必要があります。
       パソコン付属の専用起動ディスクが、Windows 98SEベースの場合、または
       Windows 98SEの｢コントロールパネル｣で作成した起動ディスクを準備できる
       場合には、本機の取扱説明書をご参照のうえ、本機用の起動ディスクを作成して
       ください。
       なお、｢1. Windows 2000を導入する前の準備｣で作成した｢Windows 2000用の
       セットアップ起動ディスク｣では、本機を起動することはできません。

       そして本機とパソコンを接続し、このCD-ROM起動ディスクでパソコンを起動
       させた後、本機にWindows 2000のCD-ROMを挿入し、次のプログラムを実行して
       ください。
           (CDドライブ):\I386\WINNT.EXE

       なお、本機に挿入するWindows 2000のCD-ROMは、(プロダクト、バージョン)
       アップグレード版でなく、通常のフルパッケージ版をお使いください。

5. 補足説明
------------

   5-1. SETUP.EXEを実行してWindows 2000をインストールする

   "2.Windows 98SE/Me上からWindows 2000をインストールする"で記述した方法以外に、
   Windows 98SE/Me上で、Windows 2000 CD-ROMのSETUP.EXEを実行してインストール
   する方法もありますが、この方法では、｢Windows 2000にアップグレードする(推奨)｣
   ことができません。Windows 98SE/MeからWindows 2000にアップグレードﾞする場合
   には、必ず、"2.Windows 98SE/Me上からWindows 2000をインストールする"で記述
   した方法に従ってください。
   なお、｢Windows 2000 を新規インストールする｣場合には、以下の手順にしたがっ
   てください。
   
    (1) プロダクト キーを入力後表示される｢特殊なオプションの選択｣画面で、
        [詳細オプション]ボタンをクリックする。
        ｢詳細オプション｣画面が表示されます。

    (2) [セットアップファイルをすべて CD からハードディスクにコピーする]に
        チェックをつけ｢OK｣ボタンをクリックし、｢次へ｣をクリックする。

    (3) 画面の指示に従って操作していくと｢インストールファイルのコピー｣が
        完了して、コンピュータが再起動します。

    (4) 以後　画面の指示に従ってインストールしてください。


   5-2. Windows 98SE/Meとデュアルブート構成する場合の注意点

         ① インストール先のドライブについて
         Windows 2000を新規インストールしてWindows 98SE/Meとデュアルブートを行う
         場合には、Windows 98SE/Meとは別のドライブに導入してください。
         同じドライブにインストールするとProgram Filesフォルダが共用となるため、
         ソフトウェアが異常動作したり、データが消失する場合もあります。

         ② ファイルシステムについて
         Windows 2000機能をフル活用するために、ファイルシステムとして
         NTFS5.0の使用が推奨されていますが、Windows 98SE/Meと併用する場合には、
         FAT/FAT32で運用するほうが互換性が高く、データの共有も容易に行えます。
         なお、Windows 98SE/MeでDriveSpaceやDoubleSpaceを、使ってドライブを圧縮
         している場合には、このドライブは、Windows 2000からは利用できませんので、
         もし利用する必要があるならば、圧縮を解除しておく必要があります。


   5-3. Windows 2000のアンインストール方法

       Windows 2000にはアンインストーラがありません。
       Windows 2000を追加インストール(デュアルブート構成)した状態で、Windows 2000
       を削除するには、以下の手順に従ってWindows 2000が格納されているディレク
       トリとブート時の選択メニューに関連するファイルを削除してください。

         ① Windows 98SE/Meの起動ディスクを使って、MS-DOSモードでパソコン起動させます。
          　｢起動ディスク｣は、Windows 98SE/Meに付属されています。もし、手元になけ
            れば、[コントロールパネル]-[アプリケーションの追加と削除]をクリックし、
            [起動ディスク]タブを選択[起動ディスクの作成]をクリックし、起動ディスク
            を作成してください。

         ② デュアルブートメニューを削除します。
            ハードディスク上のシステムブートコードを、オリジナルのWindows 98SE/Me
            用のものに修復する(書き込む)ため、以下のように入力します。
               A:\>sys a: c:
            もし、このコマンドﾞが含まれていなければ、以下のエラーメッセージが
            表示されます。
               ｢ファイルまたはコマンドが違います｣
            この場合には、C:\Windows\CommadディレクトリにあるSYSコマンドを使います。
               A:\>c:
               C:\>cd \windows\command
               C:\WINDOWS\COMMAND>sys a: c:

         ③ パソコンを再起動させます
            ハードディスク上からデュアルブートメニューが取り除かれていますので、
            すぐにWindows 98SE/Meが起動します。

         ④ 不要なWindows 2000ファイルを消去します。
            Windows 2000が含まれているフォルダをすべて削除します。
            (標準状態でインストールを行っていれば、C:\WINNTにインストールされて
            います。)
            次にC:ドライブのルートにあるWindows 2000関係のファイルも削除してく
            ださい。ただし、システム環境によっては、存在しないファイルもあります。
            また、一部のファイルは隠しファイル属性が付いていますので、｢表示｣-
            ｢フォルダオプション｣の[表示]タグで[すべてのファイルを表示する]を選択
            してください。 削除するファイル名は以下のとおりです。
               ARCSETUP.EXE
               ARCLDR.EXE
               BOOT.---
               BOOT.INI
               BOOTFONT.BIN
               BOOTSECT.DOS
               NTLDR
               NTBOOTDD.SYS
               NTDETECT.COM
               PAGEFILE.SYS
