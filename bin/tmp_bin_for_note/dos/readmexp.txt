1. Windows XPを導入する前の準備
---------------------------------

   (1) Windows XP 対応状況を確認する。

       お使いのパソコン/周辺機器のWindows XP 対応状況を、Webサイト等で確認
       してください。また、お使いのパソコン/周辺機器によっては、ドライバを
       事前ダウンロードして、フロッピーディスクなどに保存しておく必要がある
       場合があります。

   (2) 必要なデータはバックアップしておく。

       Windows XPのインストールを開始する前に、万一の場合に備えて必要なデータ
       はバックアップをとっておいてください。

   (3) BIOSを最新のバージョンにアップデートしておく。

       BIOSとは、パソコンのハードウェアを制御する基本的なソフトウェアです。
       パソコンの機種によっては、Windows XP導入時にBIOSのアップデートが必要な
       場合があります。ご使用のパソコンについて、製造元のWebサイトなどで最新の
       BIOSバージョンをご確認頂き、必要であればBIOSのアップデートを実施して
       ください。なお、BIOSのアップデートを行う前には、ヘルプ等をよく読んでから
       実施してください。

   (4) 新規インストールの場合、Windows XP用セットアップ起動ディスクを作成する。

       本機でCD-ROMブートできないパソコンにWindows XPを新規インストールする
       には、Windows XP用セットアップ起動フロッピーディスクをご使用ください。
       このセットアップ起動ディスクは、マイクロソフト社のWebサイトから
       ブートディスク作成ツールをダウンロードして、作成することができます。
       (フォーマット済み空フロッピーディスクが６枚必要です)

2. Windows XP インストール方法について
-----------------------------------------

      Windows XPをインストールするには、次の３種類の方法があります。

        ① 既存のWindowsをWindows XPにアップグレードする (上書きインストール)
           既存のWindows環境をそのまま引き継ぐインストール方式です。
           具体的には、アプリケーション、Outlook Expressのデータや 
           Internet Explorerの設定が引き継がれます。しかし、インストールに
           失敗すると、元の環境をすべて失ってしまう危険性があります。
           (Windows 2000からWindows XP Home Editionへはアップグレードでき
            ません)

        ② 既存のWindowsから、新規にWindows XPをインストールする
           既存のOS環境はそのままで、別のパーティションに新規にWindows XPを
           インストールし1台のパソコンで｢デュアルブート」させることができます。

        ③ ハードディスクをフォーマット後、新規にWindows XPをインストールする
           OSがインストールされていないパソコン、すなわちハードディスクを
           フォーマットして新規にWindows XPをインストールする方法です。

　　　== お願い ==
      ・内蔵CD-ROM/DVD-ROMドライブがあるパソコンで、Windows XPをインストール
        される場合には、なるべく本機でなく内蔵ドライブを使ってWindows XPの
        インストールを行ってください。

      == お知らせ ==
      Windows XPのファイルシステムには、FATとNTFSがあります。
      NTFSファイルシステムは、Windows 98SE/Meで互換性がありません。
      NTFSファイルシステムを使用してWindows XPをインストールした後で、パソコンを
      元の環境へリカバリする場合、ハードディスクのフォーマットをFAT形式に変更する
      必要がある場合があります。

3. 既存のWindowsをWindows XPにアップグレードする (上書きインストール)
---------------------------------------------------------------------

   既存のWindowsをWindows XPにアップグレードする場合には、以下の手順に従って
   ください。

    (1) 本機にWindows XPのCD-ROMをセットする。
        Windows XPのインストール選択画面が表示されます。

    (2) 画面の指示に従ってインストールする。

  == お知らせ ==
      Windows 98SEからWindows XPにアップグレードする場合、インストールを進めて
      いくと途中で｢アップグレードレポート｣画面が現れ、この中で本機用の
      デバイスドライバー(CDRRW10 USB CD-R/RW Adpter(ハードディスクコントローラー))
      が表示されていますが、そのまま｢次へ｣をクリックして先へ進めてください。


---------------------------------------------------------
4. 既存のWindowsから、新規にWindows XPをインストールする
---------------------------------------------------------

   既存のWindowsから、別のパーティションに新規にWindows XPをインストールする
   場合には、以下の手順に従ってください。

    (1) 既存のWindowsでパソコンを起動する。

    (2) 本機にWindows XPのCD-ROMをセットする。
        Windows XPのインストール選択画面が表示されます。

    (3) [Windows XPをインストールする]をクリックします。

    (4) [インストールの種類]で、｢新規インストール(詳細設定)｣を選択する。

        画面の指示に従って操作を進めていくと、｢ライセンス契約｣、｢プロダクト
        キー入力画面｣の後で、｢セットアップオプション｣画面が表示されます。

    (5) ｢セットアップオプション｣画面で、｢詳細オプション｣ボタンをクリックする。
        ｢詳細オプション｣画面が表示されます。

    (6) 以下の2つにチェックを付けて、[OK]をクリックする。
         □セットアップCDからすべてのインストールファイルをコピーする。
         □セットアップ中にインストールドライブ文字とパーティションを選択する。

    (7) 画面の指示に従って操作してください。

        Windows XPのインストール先パーティション(ドライブ)は、再起動が発生
        した後、背景がブルー画面のWindows XP セットアップ画面で選択できます。
　　　

-----------------------------------------------------------------------
5. ハードディスクをフォーマット後、新規にWindows XPをインストールする
-----------------------------------------------------------------------

  5.1 本機でCD-ROMブートできるパソコンの場合

    (1) 本機をパソコンに接続する。

    (2) 本機にWindows XPのCD-ROMを入れる。

    (3) パソコンの電源を入れ、CD-ROMブートさせる。
        (お知らせ: CD-ROMブートしない場合には、BIOS設定で起動の順番を確認ください)

    (4) ｢Press any key to boot from CD｣が表示されたら、何かキーを押す。

    (5) 画面の指示に従ってインストールする。



  5.2 本機でCD-ROMブートできないパソコンの場合

  == 制限事項 ==
  本機を使ってWindows XPを新規インストールするには、フロッピーディスク
  ドライブが必要です。さらに、フロッピーディスクドライブがあっても、
  外付けUSB接続タイプのフロッピーディスクドライブ場合には、パソコンに
  ２つ以上のUSBポートが必要です。

    (1) 本機とフロッピーディスクドライブをパソコンに接続する。

    (2) ｢1. Windows XPを導入する前の準備｣で作成したWindows XP用セットアップ
        起動ディスクの1枚目をフロッピーディスクドライブに挿入する。

    (3) パソコンの電源を入れ、フロッピーブートさせる。
        (お知らせ: フロッピーブートしない場合には、BIOS設定で起動の順番を確認ください)

    (4) 画面の指示に従ってインストールする。



------------
6. 補足説明
------------

      Windows XPをインストールするには、下記の方法もあります。
      ただし、この方法でも上述と同様フロッピーディスクドライブが必要です。
     
      また、ハードディスクをフォーマットする前に、ご使用のパソコンで本機が
      起動するように、起動フロッピーディスクを作成しておく必要があります。

      パソコン付属の専用起動ディスクが、Windows 98SE/Meベースの場合、または
      Windows 98SE/Meの｢コントロールパネル｣で作成した起動ディスクを準備できる
      場合には、本機の取扱説明書をご参照のうえ、本機用の起動ディスクを作成
      してください。
      なお、｢1. Windows XPを導入する前の準備｣で作成した｢Windows XP用の
      セットアップ起動ディスク｣は、この方法には使用できません。

      (1) 本機とフロッピーディスクドライブをパソコンに接続する。

      (2) 上記で作成した起動フロッピーディスクでパソコンを起動する。

      (3) smartdrv.exeが起動していない場合には、起動させてください。
          smartdrv.exeを起動せずに、以下のwinnt.exeを実行させると
          セットアップに非常に時間がかかる場合があります。

      (4) Windows XPのCD-ROM上の次のプログラムを実行する。
           (CDドライブ):\I386\WINNT.EXE

      (5) 画面の指示に従って操作する。
